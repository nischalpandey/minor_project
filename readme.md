# SURAKSHYA CAM

## Introduction
 
    The Smart Security Surveillance System is made to make security better by using
    intelligence and computer vision. Old security systems need people to watch them all the time,
    which can be a waste of time and people can make mistakes. This project is making a system that
    can find things going on, recognize people and send alerts right away.

## Features

    - Detects people in the image
    - Recognizes people
    - Sends alerts

## Technologies

    - Python
    - OpenCV
    - Django
    - SQLite
    - Yolo
    - Dlib
    - Tescreet OCR


## Installation
# Activate the virtual environment
$ source .venv/bin/activate
# Install the requirements
$ pip install -r requirements.txt
# Run the server
$ python manage.py runserver
